== Installation ==

= Installing the plugin =
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **Contact Form 7 Designer** and click "*Install now*"
2. Alternatively, download the plugin and upload the contents of `cf7-styles.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
3. Activate the plugin
4. Navigate to Form Styles and start adding your designs.

Need help? Please take a look at the FAQ section first. If you can't find your answer there, start a topic in the Support tab.
